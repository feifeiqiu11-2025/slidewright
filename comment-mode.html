01 — Demo

Comment mode turns a deck into a review surface.

Press C (or the button, top-left), then click any element or select text to leave a note.

02 — How

Point, don't retype.

Each note captures the slide number, a CSS path, and the exact text you pointed at.

03 — Handoff

Copy for agent.

All notes export as one structured Markdown bundle — paste it back to your coding agent.
